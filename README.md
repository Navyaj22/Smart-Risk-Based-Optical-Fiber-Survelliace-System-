
##  Overview

GeoSmart is a web-based GIS application developed to demonstrate intelligent last-mile connectivity planning for optical fiber networks. The project provides an interactive dashboard where users can visualize routes, compare multiple path options, estimate deployment costs, and explore AI-assisted route planning concepts.

This project was developed as part of a hackathon/academic project to address challenges in planning optical fiber connectivity in rural and underserved areas.

---

## Features

-  Interactive map using Leaflet.js
-  Select source and destination locations
-  Add intermediate (via) points
-  Compare multiple route options
-  stimated deployment cost calculation
-  Distance estimation
-  Multiple map layers (Street, Satellite, Topographic)
-  Location search functionality
-  Gram Panchayat boundary visualization (demo)
-  AI-inspired route optimization simulation

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Navyaj22/Smart-Risk-Based-Optical-Fiber-Survelliace-System-.git
   ```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

No additional installation is required.
